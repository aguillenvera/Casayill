Instalar Composer https://getcomposer.org/download/ 

1. Abra el programa Laragon, dale a iniciar .
2. darle a base de datos en la interfaz
3. Pulse el boton abrir.
4. Ahora vamos a crear una base de datos para nuestro proyecto Pulse click derecho en la parte izquierda donde hay como una lista.
5. Seleccione crear nuevo y luego base de datos y le coloca el nombre inventario.
6. Abra una terminal donde esta la aplicacion descargada (haciendo click derecho y open terminal en el explorador de archivos).
7. escriba los siguientes comandos en orden :

    _> composer install
    _> php artisan key:generate
    _> php artisan migrate --seed 

8. ahora en los iconos ocultos de la barra de navegacion de windows busque el icono de laragon y le da click derecho, arrastre el mouse hacia www 
y luego le da click en el proyecto que se llama inventario
